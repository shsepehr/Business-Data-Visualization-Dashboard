#  Business Sales Dashboard (Streamlit)

A lightweight and interactive **data visualization dashboard** built with **Streamlit**.  
This project renders product sales data and displays it in a clear bar chart to support business analysis and decision-making.  

 Perfect as a starting point for real-world dashboards, analytics interfaces, and internal business tools.

---

## Features

-  Interactive bar chart for sales analysis  
-  Simple, clean and modular code structure  
-  Business-friendly interface for non-technical users  
-  Easy to extend with CSV uploads, database connections, or real-time data  

---

## Tech Stack

| Tool | Purpose |
|------|----------|
| **Streamlit** | Web UI / Dashboard |
| **Pandas** | Data handling and transformation |
